# 僑界名片簿

僑界社團用的名片管理 PWA App。

## 功能

- 手機拍照或上傳名片照片，AI 自動辨識（支援一張照片多張名片）
- 智慧合併重複聯絡人（正反面自動偵測）
- 群組寄信（mailto BCC）
- 匯出 Google 聯絡人 CSV（自動建立「僑界名片」群組標籤）
- 前端密碼保護
- PWA 支援（可加到 iPhone 主畫面當獨立 App）

## 部署

靜態網站，部署在 Vercel。

## 設定

- 預設密碼：`hakka2026`（可在 App 設定裡修改）
- AI 名片辨識需要 Anthropic API Key（在設定裡輸入）
- 手動新增、搜尋、匯出 CSV、群組寄信不需要 API Key
